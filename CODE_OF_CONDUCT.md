# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be kind. Disagree on the merits. No harassment, no personal attacks, no bigotry. Maintainers may remove comments, commits, code, issues, and other contributions that don't align with this standard.

To report a problem, open a GitHub issue (or, for sensitive matters, contact the repository owner directly via the email listed on their GitHub profile).
